# App_client_ai

App client ai with Npm_jsMvcFw package.

## Info:

AI client application for communicating with a model hosted on a server.

## Installation

1. Write on terminal:

```
npm install
```

2. For development write on terminal:

```
npm run tauri_dev
```

3. For build write on terminal:

- 64-bit (AppImage, deb, rpm):
```
npm run tauri_build_linux_x86_64
```

- 64-bit (Exe, nsis):
```
npm run tauri_build_windows_x86_64
```

## Reset

1. Remove this from the root:

    - node_modules
    - package-lock.json

2. Follow the "Installation" instructions.
